IPython Notebooks¶

An IPython notebook is made up of a number of cells. Each cell can contain Python code. You can execute a cell by clicking on it and pressing [Shift-Enter] (for additional info please see the corresponding Wiki entry on OLAT, especially for OS dependent commands). When you do so, the code in the cell will be executed, and the output of the cell will be displayed beneath the cell. For example, after running the first cell the notebook looks like this:

In [1]:
x = 5
print(x+3)

8

Global variables are shared between cells. Therefore, we can access x in the next cell.

In [3]:
print(x+10)

15

Numpy¶

What is numpy? A library for easily performing vectorized operations on multi-dimensional data. At its core lies the numpy array.

Mathematical operations:

In [15]:
x = np.array([[1,2],[3,4]])
y = np.array([[5,6],[7,8]])

print('Elementwise sum: ')
print(np.add(x, y)) # eqivalent to print(x + y)

print('Elementwise difference: ')
print(np.subtract(x, y)) # eqivalent to print(x - y)

print('Elementwise product: ')
print(np.multiply(x, y)) # eqivalent to print(x * y)

print('Elementwise division: ')
print(np.divide(x, y)) # eqivalent to print(x / y)

Matplotlib¶
